## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Ambiente de Staging

Este projeto agora suporta execucao com um arquivo dedicado de staging.

1. Crie seu arquivo local de staging:

```bash
cp .env.staging.example .env.staging
```

2. Preencha os valores reais no `.env.staging`.

3. Rode a aplicacao com as variaveis de staging:

```bash
npm run dev:staging
```

Para build/start em modo de staging:

```bash
npm run build:staging
npm run start:staging
```

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
